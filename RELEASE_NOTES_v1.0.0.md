## üöÄ Getting Started

### Installation

```bash
pip install gravity-framework
```

### Quick Start

```python
from gravity_framework import GravityFramework

# Initialize framework
framework = GravityFramework(project_path="./my-app")

# Discover and orchestrate microservices
framework.discover_from_git("https://github.com/user/service")
framework.orchestrate_all()

# OR: Autonomous development
framework.autonomous_develop(
    description="E-commerce platform with React frontend",
    industry="e-commerce"
)
```

## üéØ Use Cases

### 1. Microservices Orchestration
```python
# Discover services from multiple repos
framework.discover_from_git("https://github.com/company/auth-service")
framework.discover_from_git("https://github.com/company/payment-service")
framework.discover_from_git("https://github.com/company/frontend")

# Orchestrate everything
framework.orchestrate_all()
```

### 2. Autonomous Development
```python
# Describe your app - AI builds it
framework.autonomous_develop(
    description="""
    E-commerce platform with:
    - React frontend with TailwindCSS
    - FastAPI backend
    - PostgreSQL database
    - Redis caching
    - Stripe payment integration
    """,
    industry="e-commerce"
)
```

### 3. Database Management
```python
# Access all microservice databases
from gravity_framework.database import MultiDatabaseAccessManager

manager = MultiDatabaseAccessManager("./services")
await manager.initialize()

# Query across services
users = await manager.execute_query("auth-service", "SELECT * FROM users")
orders = await manager.execute_query("order-service", "SELECT * FROM orders")
```

### 4. Smart Git Management
```python
from gravity_framework.git import CommitManager

manager = CommitManager(git_integration, ai_assistant)

# Smart commit with auto-categorization
result = manager.smart_commit_and_push(auto_generate=True)
```
